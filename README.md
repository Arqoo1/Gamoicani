## ✨ About

**გამოიცანი** is a modern multiplayer quiz application where players can compete in real time across multiple educational and word-based game modes.

The app combines entertainment and learning through a competitive ranking system, points, achievements, and multiplayer gameplay.

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/gamoicani.git
cd gamoicani
```

### 2. Install & Run Frontend

```bash
cd frontend
npm install
npm run android
```

### 3. Install & Run Backend

Open a new terminal.

```bash
cd backend
npm install
npm run dev
```
